
## THREE.js 核心概念
### 场景(Scene): 3D空间的容器
承载所有的3D对象(网格, 灯光, 相机, 模型等)的容器, 是THREE的空间基础, 所有需要被渲染的元素都必须添加到场景中.

```js
// 创建场景
const scene = new THREE.Scene()


// ----- 子对象管理 -----
// 添加对象到场景
scene.add(mesh, light, camera)
// 从场景移除指定对象
scene.remove(mesh)
// 清空场景所有子对象  需要确定是否会销毁内存开销
scene.clear()
// 通过 mesh.name 获取对象
scene.getObjectByName(name)
// 通过 mesh.id 获取对象
scene.getObjectById(id)


// ----- 场景树遍历 -----
// 深度优先遍历所有子对象, 包含子对象的子对象
scene.traverse(obj => {})
// 遍历场景所有可见的子对象
scene.traverseVisible(obj => {})

// ----- 背景设置 -----
// 设置背景色
scene.background = new THREE.Color(0xf0f0f0)
// 设置纹理背景
const textureLoader = new THREE.TextureLoader()
const bgTexture = textureLoader.load(imgUrl)
scene.background = bgTexture

// 设置透明背景(配合渲染器开启alpha)
scene.background = null // 清空背景
const renderer = new THREE.WebGLRenderer({alpha: true}) // 渲染器开启透明

// 天空盒子背景(6张全景图, 实现360全景环境)
const cubeTextureLoader = new THREE.CubeTextureLoader()
const skybox = cubeTextureLoader.load([img, img, img, img, img, img])
scene.background = skybox

// 添加雾化
// 雾化会作用于所有对象, 如果想让某个对象不受雾化影响, 设置 obj.fog = false
scene.fog = new THREE.Fog(oxfofofo, 10, 50) // 10单位内无雾, 50单位外完全雾化
scene.fog = new THREE.FogExp2(oxfofofo, 0.01)   // 雾化效果随距离指数增长
```


| 功能分类       | 核心 API                                  |
| :------------- | :---------------------------------------- |
| 基础创建       | new THREE.Scene()                         |
| 子对象管理     | add()、remove()、clear()、getObjectByName()|
| 场景树遍历     | traverse()、traverseVisible()              |
| 视觉配置       | background、fog、environment、overrideMaterial |
| 状态查询       | children、count、isScene                  |
| 高级操作       | localToWorld()、updateMatrixWorld()        |
| 销毁清理       | clear()、清空background/environment/fog   |

### 相机(Camera): 观察场景的视角
观察场景的视角, 视野,投影方式, 渲染器会根据相机的位置, 朝向, 参数, 将3D场景投影为2D屏幕图像.

#### 核心类型
##### PerspectiveCamera(透视相机)
透视投影, 近大远小, 有透视效果
```js
// fov 垂直视野角度, aspect 宽高比, near 近裁剪面, far 远裁剪面
const camera = new THREE.PerspectiveCamera(fov, aspect, near, far)
```

##### OrthographicCamera(正交相机)
正交投影, 无透视, 物体大小与距离无关

### 渲染器(Renderer): 将 3D 场景绘制到屏幕
### 几何体(Geometry): 物体的形状定义
### 材质(Material): 物体的外观特性
### 网格(Mesh): 几何体和材质的组合
